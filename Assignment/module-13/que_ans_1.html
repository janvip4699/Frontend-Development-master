What is Redux ?

Redux is a library of react js. A state management systems for cross-component or app-wide state. Redux is a library that is used to manage data and provides more libraries to manage its data in very simple way.

Redux is a useful solution for organising application state and provides a predictable state container for JavaScript apps. It's a well-known framework for managing state in React projects, but it also works with Angular, Vue. js, and plain old JavaScript.

10. How can we club two or more rows or columns into a single row or column in an HTML table? With Example.


To make a cell span over multiple rows, rowspan attribute is used. Rowspan Example :

Name Age
Janvi 24
Prisha


To make a cell span over multiple columns, colspan attribute is used. Colspan Example :

Name Age
Janvi Prisha 24
Riddhi Hiya 25

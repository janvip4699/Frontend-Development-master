11. What is the difference between a block-level element and an inline element?

Block-level Element Example :

Lorem ipsum dolor, sit amet consectetur adipisicing elit. Mollitia voluptatum numquam totam, voluptate itaque temporibus beatae ab, accusantium ipsam consequuntur nobis eaque, animi iste aliquid quas veritatis maxime placeat distinctio?

Lorem ipsum, dolor sit amet consectetur adipisicing elit. Quas deserunt soluta est nesciunt laboriosam molestiae id numquam cumque voluptas quis, vitae suscipit ea maiores fugiat veritatis libero asperiores aut harum!

The above p and div tags or elements are both block-level elements, & they will always start on a new line & take up the full width available.



Inline Element Example :

Lorem ipsum dolor sit amet consectetur, adipisicing elit. Delectus, consequuntur sunt! Esse expedita quam modi, Hello World!! dolorum fugit iste eum iusto possimus recusandae saepe explicabo, vero quasi optio necessitatibus voluptatibus vel?

The above highlited text is an inline span element inside a paragraph. It will not start on a new line and takes up as much width as necessary.

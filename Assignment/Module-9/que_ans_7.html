How to install React Js on Windows, Linux Operating System? How to Install NPM and How to check version of NPM?

Steps to install React JS on windows : 1. download node js in your systems 2. install node js 3. check verson of node js and npm and npx node -v npm -v Steps to install React JS on Linux : 1. Update the system sudo apt update -y && sudo apt upgrade -y 2. Install Node JS sudo apt install nodejs -y 3. Install NPM sudo apt install npm -y 4. Install React JS npm install -g create-react-app

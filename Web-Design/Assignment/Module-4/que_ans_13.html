13. What are the ways to integrate CSS as a web page?

We can integrate CSS as a web page in three different ways. To style a single HTML element on the page, use inline CSS in a style attribute.

By adding CSS to the head section of the HTML page, we can embed an internal stylesheet & we can also add external stylesheet that separates our CSS from our HTML.

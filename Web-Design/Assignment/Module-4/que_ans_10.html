10. Why should background and color be used as separate properties?

By having separate properties, we can easily change the background and text color independently & it also provides flexibility and specificity.

Background specifies all the background properties like background-color, background-image, background-position, background-attachment etc. & color specifies the color of the HTML element.(text color)

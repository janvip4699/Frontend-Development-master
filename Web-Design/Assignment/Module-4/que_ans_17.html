17. What is the meaning of the CSS selector?

A CSS selector selects the HTML elements you want to style. There are five categories of CSS selectors :

CSS element selector : It selects HTML elements based on the element name.

CSS ID selector : It uses the id attribute of an HTML element to select a specific element. The id of an element is unique within the page so it is used to select one unique element. It is hash(#) followed by id of the element.

CSS class selector : It selects HTML elements with a specific class atribute. It is period(.) followed by the class name.

CSS universal selector : It selects all HTML elements on the page.(*)

CSS grouping selector : It selects all the HTML elements with the same style definitions. To apply same stylesheet to multiple different elements you can group the elements. It will help to minimize the code.
